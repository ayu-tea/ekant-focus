# Ekant

Ekant is a minimal focus timer built for deep work, study sessions, and distraction-free productivity.

Unlike many productivity applications, Ekant focuses on simplicity. There are no accounts, databases, cloud synchronization, achievements, or social features—just the tools needed to focus.

## Features

- Countdown timer (1–180 minutes)
- Stopwatch mode
- Animated progress ring
- Zen Mode
- Keyboard shortcuts
- Theme customization
- Dark Mode

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Philosophy

> Open. Focus. Work.

Ekant was created as a lightweight alternative to feature-heavy productivity apps. The goal is to provide a calm environment that helps users focus on their work without introducing additional distractions.
